# E-commerce Sales Analytics Dashboard

A sales analytics dashboard built with SQL Server and Power BI, analyzing e-commerce sales data sourced from Kaggle.

## Project Structure

```
├── sql/                 # All SQL scripts (CREATE, queries, stored procedures)
├── powerbi/             # Power BI .pbix report file
├── images/
│   └── dashboard_screenshots/  # Screenshots of the dashboard
└── docs/                # Any additional documentation
```

## Setup

1. Run the script in `sql/` to build the schema, load data, and create stored procedures
2. Open the `.pbix` file in `powerbi/` with Power BI Desktop and update the data source connection

> Note: Power BI Desktop (free) is required to open the `.pbix` file. Publishing to the web requires a Power BI Pro license.
